## Key UI Elements

### Create Layout Mode

1. **Layout Label Input**: Text field for naming the bind group layout
2. **Binding Entries Table**: 
   - Shows all defined bindings
   - Columns: Binding number, Type, Visibility checkboxes, Delete button
   - Checkboxes for Vertex/Fragment/Compute stage visibility
3. **Add Binding Buttons**: Six buttons for different binding types
4. **Action Buttons**: Validate and Reset

### Bind Resources Mode

1. **Bind Group Label Input**: Text field for naming the bind group instance
2. **Layout Summary Box**: Shows the selected layout name and binding count
3. **Resource Assignment Cards**: One per binding slot
   - Shows binding number and expected type
   - Displays current assignment status
   - Lists all compatible resources as clickable buttons
   - Resources show relevant details (size, format, dimensions, etc.)
4. **Action Buttons**: Validate Bindings and Create Bind Group

- **Selected Items**: Highlighted buttons with border
- **Unassigned Bindings**: Warning indicator
