## Interactive Elements

1. **Checkboxes**: Enable/disable features
   - Color attachment
   - Depth-stencil attachment
   - Timestamp writes

2. **Text Inputs**:
   - Label (single line)
   - Timestamp beginning index
   - Timestamp end index

3. **Sliders**:
   - Clear color R, G, B, A (0.0 to 1.0)
   - Clear depth (0.0 to 1.0)
   - Clear stencil (0 to 255)

4. **Dropdown Menus**:
   - Load operation (Clear/Load)
   - Store operation (Store/Discard)

5. **Buttons**:
   - Validate
   - Reset
   - Preset: Black Clear
   - Preset: With Depth

6. **Color Preview**: Visual representation of clear color

7. **Hover Tooltips**: Explanatory text on hover
   - Load operation: "What to do at the start of the render pass"
   - Store operation: "What to do at the end of the render pass"
   - Beginning Index: "Query set index for the beginning timestamp"
   - End Index: "Query set index for the end timestamp"
