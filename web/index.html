About this demo

This is an interactive WebGPU demo built with Rust and wgpu. It showcases GPU rendering capabilities directly in your browser using WebAssembly.

Requirements: A browser with WebGPU support (Chrome 113+, Edge 113+, Safari Technology Preview*, or Firefox Nightly*).

* Requires manual enablement in browser settings
